## 🔧 技术栈
- 开发引擎：Unity 2022.3.53f1c1
- 开发语言：C#
- 核心技术：对象池(OP)、有限状态机(FSM)、行为树(Behavior Tree)、UI框架搭建、PlayerPrefs数据持久化

## 📖 项目简介
软件工程专业本科毕业设计，基于Unity引擎实现2D横版动作角色扮演游戏（ARPG）核心功能。聚焦游戏性能优化与NPC智能AI逻辑实现，通过工程化的开发思路解决游戏实体管理、代码耦合性、数据存储等核心问题

## ✨ 核心功能与技术实现
### 游戏实体管理：对象池（Object Pool）
- 设计并实现对象池模式，统一管理游戏中高频生成/销毁的实体（如怪物、道具、特效）；
- 避免频繁创建/销毁对象带来的内存开销，**显著降低系统资源占用**，提升游戏运行流畅度。

### NPC智能AI：有限状态机(FSM) + 行为树
- 基于有限状态机实现NPC基础状态切换（移动、攻击、巡逻、受伤、死亡等），有效降低实体代码间的耦合性；
- 结合行为树构建NPC智能决策系统，支持6-8种复杂行为逻辑（如目标追踪、仇恨判定、技能释放、巡逻路径规划等）；
- 状态机与行为树结合的设计，兼顾AI逻辑的灵活性与代码的可维护性。

### UI框架与数据持久化
- 搭建高复用性UI框架，实现界面层级管理、弹窗逻辑、异步加载等核心功能，适配不同分辨率；
- 基于PlayerPrefs实现游戏进度、玩家数据（等级、装备、存档）的持久化存储，保障数据不丢失。

> 本项目为毕业设计作品，聚焦游戏开发中的工程化思想与性能优化方向，所有核心功能均独立设计与实现。
